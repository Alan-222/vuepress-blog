    vue3后台管理系统-角色管理页面🔥

    龙二 2023/1/22 vue3tsnode.jsvue3后台管理系统

    # 前言

    完成了用户管理页面，剩下两个页面角色管理以及菜单管理也大同小异。但角色管理还有一个分配资源的功能需要实现，作用为选择角色的权限，我们可以通过el-tree树形控件来完成。当打开分配角色弹窗时给树形控件传递一个树形结构的数组去渲染完整的权限结构（菜单+对应的按钮） ，然后获取角色的权限 id勾选就可以。然后更新角色权限只需要改变权限时记录下权限 id再调用更新角色资源的接口即可。

    # 角色管理分配资源的实现

    # 接口数据接口

    首先我们测试一下获取菜单项接口大概得到怎样的菜单结构，如图

    数组结构的话大概就是

    {
  label:"xxx",   // 菜单标题
  value:"xxx"    // 权限id
  children:[{    // 子孩子数组
    label:'xx',
    value:'xx',
    perms:[{      // 按钮数组
      label:"xx"
      value:''
    }]
  }]  
}

    符合el-tree树形控件的要求，value属性作为每个树节点用来作为唯一标识的属性

    # html 结构

    我们首先确定html结构，用el-scrollbar包裹树形控件，超出长度显示滚动条。然后把他们放在弹窗里，点击分配资源按钮显示弹窗。树形控件中用到插槽，包括菜单的标题，若有按钮的话则显示复选框（遍历菜单所有的按钮）

    js 代码

    上面的 html 结构的el-tree中有@check-change方法，能获得到勾选菜单的数据和当前勾选的状态，当勾选或者取消菜单选项时触发。我们只要在勾选菜单时将对应按钮checkbox的状态改为跟当前菜单勾选状态一样即可。我用了btnPerms的数组来存储按钮，结构如下

    比如当我取消勾选了图中 20、21 的父菜单，就将 20、21 的checkbox也取消勾选。代码如下：

    // 树形控件菜单勾/反选方法
const handleResourceCheckChange = (
  data: Resource,
  isCheck: boolean,
  sonHasCheck: boolean
) => {
  console.log('data', data);
  console.log('isCheck', isCheck);
  // 当勾/反选的菜单存在按钮时，按钮的状态改成对应菜单状态
  if (data.perms) {
    data.perms.forEach(item => {
      btnPerms.value[item.value] = isCheck;
    });
  }
};

    还有就是当打开角色分配资源弹窗时候的方法，有以下几个步骤。

    • 获取全部的菜单结构
    • 获取该角色所有的菜单 id 数组和按钮 id 数组，对应进行勾选

    方法如下所示：

    /**
 * 资源分配

function openRoleResourceDialog(row: RoleItem) {
  resourceDialogVisible.value = true;
  loading.value = true;
​
  const roleId: number = row.role_id;
  checkedRole.value = {
    role_id: roleId,
    role_name: row.role_name
  };
​
  // 获取所有的资源
  listMenuOptions().then(response => {
    resourceOptions.value = response.data;
​
    // 获取角色拥有的资源
    getRoleResources(roleId).then(({ data }) => {
      // 勾选的菜单回显
      const checkedMenuIds = data.menu_ids;
      resourceRef.value.setCheckedKeys(checkedMenuIds);
​
      nextTick(() => {
        // 勾选的权限回显
        const rolePermIds = data.permIds;
​
        state.allPermIds = filterResourcePermIds(response.data, []);
        if (state.allPermIds) {
          state.allPermIds.forEach(permId => {
            if (rolePermIds.indexOf(permId) > -1) {
              btnPerms.value[permId] = true;
            } else {
              btnPerms.value[permId] = false;
            }
          });
        }
        loading.value = false;
      });
    });
  });
}
​
const filterResourcePermIds = (resources: Resource[], permIds: string[]) => {
  resources.forEach(resource => {
    if (resource.perms) {
      resource.perms.forEach(perm => {
        permIds.push(perm.value);
      });
    }
    if (resource.children) {
      filterResourcePermIds(resource.children, permIds);
    }
  });
  return permIds;
};

    然后是提交资源的方法。我们可以通过el-tree控件的getCheckedNodes的方法获得勾选的菜单数组，再通过上面的btnPerms获得勾选的按钮数组。然后凭这两个数组更新角色资源。

    /**
 * 分配资源提交
 */
function handleRoleResourceSubmit() {
  const checkedMenuIds: any[] = resourceRef.value
    .getCheckedNodes(false, true)
    .map((node: any) => node.value);
​
  const checkedPermIds = [] as string[];
  if (state.allPermIds) {
    state.allPermIds.forEach(permId => {
      if (btnPerms.value[permId]) {
        checkedPermIds.push(permId);
      }
    });
  }
​
  const RoleResource = {
    menu_ids: checkedMenuIds,
    permIds: checkedPermIds
  };
​
  updateRoleResource(checkedRole.value.role_id, RoleResource).then(res => {
    ElMessage.success('分配权限成功');
    state.resourceDialogVisible = false;
    handleQuery();
  });
}
